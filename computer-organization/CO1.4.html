运算速度：

影响计算机的运算速度有很多技术指标，主要包括： 吞吐量, 响应时间, 主频, CPU时钟周期, CPI, CPU执行时间, MIPS, MFLOPS,

吞吐量：

指计算机系统在单位时间内处理请求的数量

响应时间

指计算机系统对请求做出响应的时间，响应时间包括CPU时间与等待时间的总和

主频：

又称时钟频率，表示在CPU内数字脉冲信号振荡的速度

时钟周期：

CPU主频的倒数，是CPU中最小的时间元素。每个动作至少需要一个时钟周期。CPI（Cycle per Instruction）：

是指每条指令执行所用的时钟周期数。在现代高性能计算机中，由于采用各种并行技术，使指令执行高度并行化，常常是一个系统时钟周期可以处理若干条指令。所以CPI参数经常用IPC（Instruction per Cycle）表示，即每个时钟周期执行的指令数：

CPI = 1 / IPC

CPU执行时间：

CPU执行时间 = CPU时钟周期数 / 时钟频率 = 指令数*CPI / 时钟频率

MIPS（Million Instructions per Second）：

表示每秒执行多少百万条指令。对于一个给定的程序，MIPS定义为：

MIPS = 指令条数 / 执行时间 * 10^6 = 主频 / CPI = 主频 * IPC

MFLOPS（Million Floating-point Operations per Second）

表示每秒执行多少百万次浮点运算，对于一个给定的程序，MFLOPS定义为：

MFLOPS = 浮点操作次数 / 执行时间 * 10^6
