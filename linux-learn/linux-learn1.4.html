浏览目录命令

命令名称：ls
命令英文原名：list
语法：ls [-ald] [文件或目录]
功能描述：显示目录文件
选项：

-a 显示所有文件，包括隐藏文件（.开头）

-l 显示详细信息

-d 查看目录属性（d/ 目录，f/ 文件）


命令名称：pwd
命令英文原名：print working directory
语法：pwd
功能描述：显示当前所在的工作目录


目录操作命令

命令名称：cd
命令英文原名：change directory
语法：cd [目录]
功能描述：切换目录
范例：
  • $ cd / 切换到根目录
  • $ cd .. 切换到上一级目录

命令名称：mkdir
命令英文原名：make directory
语法：mkdir [目录名]
功能描述：创建新目录
范例：$ mkdir newdir

命令名称：rmdir
命令英文原名：remove directory
语法：rmdir [目录名]
功能描述：删除空目录
范例：$ rmdir newdir

浏览文件命令

命令名称：cat
命令英文原名：concatenate and display files
语法：cat [-En] [文件名]
功能描述：连接文件并打印到标准输出设备上
范例：
  • cat m1 (在屏幕上显示文件m1的内容) cat m1 m2
  • (同时显示文件m1和m2的内容)

命令名称： more
命令英文原名： more
语法： more [文件名]
功能描述： 分页显示文件内容，

空格或f 显示下一页

enter 显示下一行

q或Q 退出

范例： $ more /etc/services

命令名称： less
命令英文原名： less
语法： less [文件名]
功能描述： 分页显示文件内容，除了more的操作意外还可以利用上下键来卷动文件
范例： $ less /etc/services

命令名称： head
命令英文原名： head
语法： head [-n] [文件名]
功能描述： 显示文件前几行的内容
选项： -n: 显示前n行，默认为10
范例： $ head /etc/passwd

命令名称： tail
命令英文原名： tail
语法： tail [+-n/-F] [文件名]
功能描述： 显示文件后几行的内容
选项：
  • -n: 显示后n行，默认为10
  • +n: 从第n行显示到文件尾
  • -F: 用于追踪显示不断增长的文件结尾内容（通常用于显示日志文件）
范例： $ tail /etc/passwd

文件操作命令

命令名称： cp
命令英文原名： copy
语法： cp [-piru] 源文件路径 目标文件路径
功能描述： 将文件从一处复制到另一处
选项：
  • -p: 连同文件属性一起复制，常用与备份
  • -i: 若目标文件已存在，会线询问是否覆盖
  • -r: 递归持续复制，用于目录的复制
  • -u: 目标文件与源文件有差异时才会复制
范例：
  • $ cp test1.txt test2.txt
  • $ cp test3.txt /tmp 将test3.txt复制到/tmp目录中
  • $ cp -r test1(目录) test2(目录) 拷贝目录test1到目录test2

命令名称： rm
命令英文原名： remove
语法： rm [-fir] [文件名]
功能描述：
删除文件或目录
选项：
  • -f: force，忽略不存在的文件，不会出现警告消息
  • -i: 互动模式，在删除前会询问用户是否操作
  • -r: 递归删除，最常用于目录删除，它是一个非常危险的参数
范例：
  • $ rm myfiles
  • $ rm * 删除当前目录下的所有文件
  • $ rm -r myfolder 删除某个目录
  • $ rm -f *.txt 强迫删除所有后缀名为.txt文件
  • $ rm -i * 删除当前目录下的所以偶文件，删除前会询问

命令名称： find
命令英文原名： find
语法： find 路径 [参数]
选项和参数：
  • -name filename: 找出文件名为filename的文件
  • -size [+-]SIZE: 找出比SIZE大(+)或小(-)的文件
  • -type TYPE: 查找文件的类型为TYPE的文件，TYPE的值主要有：一般文件(f)、设备而文件(b、c)、目录(d)、连接文件(l)、socket(s)、FIFO管道文件(p)
  • -perm mode: 查找文件权限刚好等于mode的文件，mode用数字表示，如0755
功能描述： 寻找文件或目录
范例：
  • $ find -name ctime 在当前目录下查找名为ctime的文件
  • $ find -type c 在当前目录下查找设备文件

命令名称： grep
命令英文原名： Globally search a Regular Expression and Print
语法： grep [-aciv] 要找的字串 要寻找的字串的源文件
功能描述： 全局，在文件中搜索匹配的字符并输出，每个输出一行
选项：
  • -a: 将binary文件以text文件的方式查找数据
  • -c: 计算找到“查找字符串”的次数
  • -i: 忽略大小写的区别，即把大小写是为相同
  • -v: 反响选择，即显示出不包‘查找字符串’内容的那一行
范例：
  • $ grep errno errno.h 找errno.h里查找errno字符串
  • $ gerp errno *.h 在所有的.h文件里查找errno字符串

命令名称： tar
命令英文原名： tape archive
语法： tar [参数] 文件
功能描述： 能够将用户所指定的文件或目录打包成一个文件，也可以通过指定参数开启压缩/解压缩功能
选项：
  • -c: 新建打包文件
  • -t: 查看打包文件的内容有哪些文件名
  • -x: 解打包或解压缩的功能，可以搭配-C(大写)指定解压的目录，注意-c,-t,-x不能同时出现在同一条命令中
  • -j: 通过bzip2支持进行压缩/解压缩
  • -z: 通过gzip的支持进行压缩/解压缩
  • -v: 在压缩/解压缩过程中，将正在处理的文件名显示出来
  • -f filename: filename为要处理的文件
  • -C dir: 指定压缩/解压缩的目录dir
范例：
  • $ tar -cvf test.tar * 将当前目录的所有文件打包成test.tar
  • $ tar -czvf test.tar.gz * 将当前目录的所有文件打包成test.tar，再用gzip命令压缩
  • $ tar -tf test.tar 查看test.tar文件中包括了那些文件
  • $ tar -xvf test.tar 将test.tar解打包
  • $ tar -xzvf foo.tar.gz 将foo.tar.gz解压缩并解打包
